What is Project Mapper?

Project Mapper is a tool designed to analyze Python projects and generate structured maps of their code and documentation components. The system uses a pipeline architecture optimized for AI agent consumption within VSCode-based IDEs, particularly Cursor.

Key Features:
  • Analyze Python code and documentation files
  • Detect relationships between components
  • Generate AI-optimized maps in various formats
  • Visualize project structure and relationships
  • Optimize token usage for AI consumption
